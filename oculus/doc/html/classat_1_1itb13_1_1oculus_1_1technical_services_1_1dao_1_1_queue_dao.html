Queue at.itb13.oculus.technicalServices.dao.QueueDao.findById ( Integer  doctorId,
Integer  orthoptistId 
)

Opens a new Hibernate session and looks up the Queue with the specified doctorID and orthoptistID. The conversion between the database table "Queue" and the at.itb13.oculus.domain.Queues with their at.itb13.oculus.domain.QueueEntrys is done automatically.

If both the doctorID and orthoptistID is null, the general orthoptist queue is returned.

Parameters
doctorIdthe id of the queue's doctor. May be null.
orthoptistIdthe id of the queue's orthoptist. May be null.
Returns
the desired queue if found, or null otherwise
